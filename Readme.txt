""STEP-BY-STEP HOW TO USE THIS PROGRAM""
1. DOWNLOAD AND INSTALL PYTHON 3.8.4
2. DOWNLOAD AND INSTALL ARDUINO IDE
3. GO TO POWERSHELL OF THIS FOLDER THEN TYPE 'pip install -r requirements.txt'
4. INSERT ANY ARDUINO HARDWARE TYPE TO PC/LAPTOP AND RUN PROGRAN OF HGR_ARDUINO.ino IN ARDUINO IDE
5. RUN PROGRAM OF HGR.py in any IDE software

NOTE: MUST BE CAMERA CONNECTION
